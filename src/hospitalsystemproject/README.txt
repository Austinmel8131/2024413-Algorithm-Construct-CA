
# Hospital System Prototype

This Java project models a hospital organization with classes for Employees, Managers, and Departments.

## Features
- Recursive Bubble Sort
- Linear Search
- Random employee generation
- ENUM-driven menu system
- Manager and department assignments

## How to Run
Compile all `.java` files and run `HospitalSystem`:
```
javac *.java
java HospitalSystem
```

## Example Commands
- Add employee → Provide name, ID, manager type, department
- Generate random → Choose how many random employees to add
- Search → Enter a name to find employees
- Sort → Alphabetically sort and display top 20 employees
